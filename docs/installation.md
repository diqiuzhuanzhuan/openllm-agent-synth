# Installation

## Stable release

To install `openllm-agent-synth`, run this command in your terminal:

```sh
uv add openllm-agent-synth
```

Or if you prefer to use `pip`:

```sh
pip install openllm-agent-synth
```

## From source

The source files for `openllm-agent-synth` can be downloaded from the [GitHub repo](https://github.com/diqiuzhuanzhuan/openllm-agent-synth).

You can either clone the public repository:

```sh
git clone https://github.com/diqiuzhuanzhuan/openllm-agent-synth.git
```

Or download the [tarball](https://github.com/diqiuzhuanzhuan/openllm-agent-synth/tarball/master):

```sh
curl -OJL https://github.com/diqiuzhuanzhuan/openllm-agent-synth/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd openllm-agent-synth
uv pip install .
```
